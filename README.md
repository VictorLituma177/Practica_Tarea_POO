Este proyecto implementa un sistema de gestión de contenido audiovisual completamente estructurado bajo buenas prácticas de programación, arquitectura limpia y principios de diseño modernos. El objetivo es administrar películas, series, documentales, videos de YouTube y cortometrajes, permitiendo cargarlos desde archivos CSV, listarlos, buscarlos y gestionarlos a través de un sistema modular y fácil de mantener.
El sistema ha sido desarrollado aplicando:
 Programación orientada a objetos

 Principios SOLID
 Arquitectura MVC
 Código limpio y refactorización
 Manejo de archivos CSV
 Pruebas unitarias con JUnit 5

El sistema permite:
Cargar películas, series y documentales desde archivos CSV.
Registrar contenido audiovisual en memoria.
Listar todo el contenido cargado.
Buscar contenido por título.
Mostrar detalles completos de cada tipo de contenido (actores, temporadas, investigadores, etc.).
Ejecutar el programa desde consola, siguiendo un menú interactivo.
Probar sus funcionalidades mediante pruebas unitarias.
